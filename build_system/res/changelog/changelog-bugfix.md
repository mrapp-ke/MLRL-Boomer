# Quality-of-Life Improvements

- Pre-built packages are now provided for Windows ARM64 systems.
- In addition to pre-built packages for Linux distributions following the [manylinux](https://packaging.python.org/en/latest/specifications/platform-compatibility-tags/#manylinux) standard, packages are now also provided for [musllinux](https://packaging.python.org/en/latest/specifications/platform-compatibility-tags/#musllinux) systems.
